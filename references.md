**2-opt swap algorithm**



The 2-opt algorithm was originally described by:



Croes, G.A. (1958). A method for solving traveling salesman problems. Operations Research, 6(6), pp.791–812.



The practical implementation used here follows the standard description in:



Applegate, D., Bixby, R., Chvátal, V. and Cook, W. (2006). The Traveling Salesman Problem: A Computational Study. Princeton University Press.



A good freely accessible description is on the Wikipedia article:



Wikipedia (2024). 2-opt. Available at: https://en.wikipedia.org/wiki/2-opt

**Haversine formula**



The formula itself is a standard trigonometric identity dating to the 19th century. The modern programming reference is:



Sinnott, R.W. (1984). Virtues of the Haversine. Sky and Telescope, 68(2), p.159.

A practical implementation reference: Movable Type Scripts (2024). Calculate distance, bearing and more between Latitude/Longitude points.
